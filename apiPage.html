Getting Started

To get started with using the FileManagerJS API, please include the following files:

FileManager.js: This is the main file that includes all the functionality aspects found in FileManagerJS.

style.css: This is the main CSS file for the FileManagerJS library.

JQuery: This is the JQuery library script needed by the library.

home.js (Optional): This file adds all user interaction components to the FileManagerJS library.

Importing this file is not necessary if the UI is not required as part of the application.

To load all 4 files: Please use the following the format below in the head of the HTML.

Note: Please call the function: InitFileManagerJS() in your JS file as the first step to initialize the library.
