FileManagerJS

Created by: Kevin Gao

FileManagerJS is a JavaScript library that enables developers to implement an attractive front-end file management layout seamlessly.

The library includes the optional addition, through input fields and buttons, of user interactive components.

Scripts To Include for Getting Started

To get started with using the FileManagerJS library, please include the following files:

FileManager.js: Below is the main file that includes all the functionality aspects found in FileManagerJS.

<script defer type="text/javascript" src='FileManager.js'></script>

Note: Please call the function: initFileManagerJS() in your JS file as the first step to initialize all the functionalities after importing the library.

style.css: Below is the main CSS file for the FileManagerJS library.

<link rel="stylesheet" type="text/css" href="./style.css">

JQuery: Below is the JQuery library script needed by the library.

<script defer src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>

home.js (Optional): Below, this file adds all user interaction components to the FileManagerJS library.

Importing this file is not necessary if the UI is not required as part of the application.

<script defer type="text/javascript" src='home.js'></script>

To load all 4 files: Please use the following the format below in the head of the HTML.

<!--- Load in JQuery ---> <script defer src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script> <!--- Load in the File Manager CSS file --> <link rel="stylesheet" type="text/css" href="./style.css"> <!--- Load in HomeJS --> <script defer type="text/javascript" src='home.js'></script> <!--- Load in FileManagerJS --> <script defer type="text/javascript" src='FileManager.js'></script>

Code Snippet to Get Started

In plain text:

//Initializes the Library initFileManagerJS(); function createStartingExample() { //Creates 3 Folder Objects const Folder1 = FolderCreator('Folder1', 'root'); const Folder2 = FolderCreator('Folder2', 'root'); const Folder3 = FolderCreator('Folder3', 'root'); //Creates 3 File Objects const File1 = new FileCreator( 'File1.jpeg', 75655, 'Picture1', 'Folder1', 'green' ); const File2 = new FileCreator( 'File2.txt', 12032, 'Report1', 'Folder2', 'red' ); const File3 = new FileCreator( 'File3.pdf', 8513, 'Document1', 'Folder3', 'purple' ); //Adds both the Folder and File Objects to the DOM addNestedFolder(Folder1); addNestedFolder(Folder3); addNestedFolder(Folder2); addFile(File1); addFile(File2); addFile(File3); //Rename a File name to 'PictureFile' renameElement(File1.userSpecifiedName, 'PictureFile'); //Move File 2 into Folder 3 moveElements(File2.userSpecifiedName, Folder3.name); } createStartingExample();
