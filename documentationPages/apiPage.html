Object Constructor API Functions

Below are all the Object Constructor functions.

FileCreator(fileName, size, userSpecifiedName, folderId, color)

Parameters:

fileName: The name of the file.

size: The size of the file.

userSpecificName: The user specified name of the file.

folderId: The name of the folder where the file will be stored into.

color: The desired color of the file. If no color is specified, it will default to the color black.

Return Type:

A File Object.

Description:

This function creates a File Object based on the specified parameter values.

FolderCreator(name, parentFolderName)

Parameters:

name: The name of the folder.

parentFolderName: The name of the folder where this new folder will be stored. The top level direction is known as 'root'.

Return Type:

A Folder Object.

Description:

This function creates a Folder Object based on the specified parameter values.


DOM Manipulation API Functions

Below are all the DOM Manipulation functions.

initFileManagerJS()

Parameters:

None.

Return Type:

void.

Description:

This function initializes all the functionality components in FileManagerJS and needs to to be called before calling other functions in the library.

addFile(File)

Parameters:

File: The File object that was created using the FileCreator function.

Return Type:

void.

Description:

This function takes in an existing File object, adds this object to the global files array, and displays a file's name, the user specified name, and the specified color in the front-end.

addNestedFolder(Folder)

Parameters:

Folder: The Folder object that was created using the FolderCreator function.

Return Type:

void.

Description:

This function takes in an existing Folder object, adds this object to the global folders array, and displays a folder's name in the front-end.

renameElement(oldElementName, newElementName)

Parameters:

oldElementName: The name of the existing file or folder.

newElementName: The new name of the file or folder

Return Type:

boolean.

Description:

This function takes the name of an existing file or folder and modifies the name to a new name, as specified in the newElementName parameter.

This function will also modify either the folder or file name in their respective arrays to the new name.

deleteElement(userElementName)

Parameters:

userElementName: The name to select the existing file or folder for deletion.

Return Type:

boolean.

Description:

This function takes the name of an existing file or folder and deletes that Object from both the DOM and its respective arrays.

changeFileColor(fileName, newColor)

Parameters:

fileName: The name of the existing file.

newColor: The new color name.

Return Type:

void.

Description:

This function takes the name of an existing file and modifies its color from both the DOM and its respective arrays to the newColor parameter value.

Note: If an invalid color name is given, the color will not be modified.

moveElements(elementName, folderName)

Parameters:

elementName: The name of the existing file or folder.

folderName: The name of the existing destination folder.

Return Type:

void.

Description:

This function takes the name of an existing file or folder and moves that Object to the new folder on the DOM.

Note: The new folder must already exist on the DOM and in the folders array.
